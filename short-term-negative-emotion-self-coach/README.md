# 短期负面情绪自我教练 Skill

这一套发布包包含两个版本：

- `default`
  - 更通用、更柔和、更适合广泛测试
- `yongcheng-style`
  - 保留永澄式的判断感、解释系统敏感度和推进力

它们都服务于同一个目标：

- 先接住
- 再澄清
- 再推进

不是为了把情绪“聊完”，
而是为了把用户从短期负面情绪里重新带回现实行动。

## 目录

- `packages/default`
- `packages/yongcheng-style`
- `TEST-CASES.md`
- `INSTALL.md`
- `dist/`

## 适用问题

- 比较后自卑
- 受挫后焦虑
- 没结果后的自我否定
- 副业 / 创业新手的短期情绪下坠

## 不适用问题

- 高危心理危机
- 临床心理治疗
- 长期深层疗愈替代

## 当前推荐

如果你第一次试：

- 先用 `default`

如果你已经熟悉永澄式表达，希望更有判断力、更少“空安慰”：

- 再试 `yongcheng-style`
